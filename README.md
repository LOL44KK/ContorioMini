## Управление
- **W, A, S, D** — передвижение персонажа.
- **B** — включить/выключить режим строительства.
- **Enter** — поставить блок.
- **E** — убрать блок.
- **Стрелки (Вверх, Вниз, Влево, Вправо)** — перемещение по спискам.
- **TAB** — Меню токенов, исследований и поиска планет.
